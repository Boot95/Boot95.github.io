Bootstarp常用类

排版

标题

标题里面的class有h1~h6来表示标题的样式。<small>是小标题的样式class。

Bootstarp定义的全局font-size是14px，line-hieght是20px，这些样式应用到了<body>和所有样式上。

可以通过添加.lead让段落突出显示。

强调

<small> :对于不需要强调的inline或者block类型的文本使用<small>标签。

加粗

<strong>用来增加font-weight值的方式加粗强调一段文本。

斜体

<em>用来把字体倾斜来强调文本的显示。

对齐类

简单方便的文本对齐

强调类

下面的class是通过颜色来强调文本的显示

缩略语

地址

<address>

引用

默认引用使用<blockquote>。把需要引用的文字放在<blockquote></blockquote>标签中就可以了。当然还是建议再用<p>标签包裹一下。

引用选项

可以用<small>来注明引用来源，来源名称放在<cite>标签里面。

另一种展示风格

使用pull-right类，可以让引用展现在右侧对齐的效果

列表

无序列表

有序列表

无样式列表

无样式列表默认移除了list-style样式

行内列表

使用inline-block让列表水平排列在同一行，同时每一行都有少量的内补（padding）

描述

对一个列表/条目进行相关联的描述

垂直描述

水平描述

水平描述就是给<dl>添加了一个class='dl-horizontal'样式。

代码

注意：使用行内代码/基本代码块的时候html左右标签包裹必须转义

行内代码

使用<code>展示行内代码。

基本代码块

使用<pre>展示多行代码。

表格

默认样式

为<table>标签增加基本样式–很少的内补(padding)并只增加水平分隔线–只要为其增加 .table 类即可。

表格可选的类

.table-striped斑马纹样式

.table-bordered为表格添加边框

.table-hover为表格添加悬停效果

.table-condensed使表格更加的紧凑

可选的属性

Class 涵义
.success 表示成功或者积极的意义
.error 表示一个危险或者存在隐患的行为
.warning 表示警告，可能需要注意
.info 作为一个默认样式的一个替代样式

受支持的表格标签

标签 描述
<table> 包裹一表格形式展示数据的元素
<thead> 包含表头<tr>的容器
<tbody> 包含表格内容<tr>的容器
<tr> 单元格tr/td的容器
<td> 默认的单元格
<th> 每行/列，（依赖于放置的位置，）所对应的label
<caption> 用于对表的描述或者是总结，对屏幕阅读器特别有用
